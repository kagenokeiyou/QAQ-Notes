# React

React 是一个免费开源的前端 JavaScript 库，由 Meta（前 Facebook）以及一群个人开发者和公司共同维护，用于基于UI组件构建用户界面

## JSX

JSX（有时也被称为 JavaScript XML）是 JavaScript 语言语法的 XML 样式的扩展，最初由 Facebook 为 React 的使用而创建，现已被多个 Web 框架采用

```jsx
const App = () => {
   return (
     <div>
       <p>Header</p>
       <p>Content</p>
       <p>Footer</p>
     </div>
   ); 
}
```
